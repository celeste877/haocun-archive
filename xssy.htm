SPARQL 查询示例

以下为示例查询，展示如何通过语义查询获取知识：

查询1：刘浩存合作过的导演
PREFIX ex: <http://example.org/movie#>
SELECT ?director WHERE {
  ex:刘浩存 ex:合作导演 ?director .
}
查询2：2020年后上映且评分高于7.0的电影
PREFIX ex: <http://example.org/movie#>
SELECT ?film ?rating WHERE {
  ?film ex:上映年份 ?year .
  ?film ex:豆瓣评分 ?rating .
  FILTER(?year >= 2020 && ?rating > 7.0)
}
