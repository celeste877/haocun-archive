# 刘浩存电影数字档案馆

基于语义网技术的电影数字档案馆，以演员刘浩存为切入点，展示电影本体、关联数据、知识图谱和 SPARQL 查询示例。

## 功能模块
- 人物档案：演艺生涯、基本信息
- 作品库：电影作品列表、详情模态框、在线观看链接
- 荣誉奖项：历年获奖与提名数据（动态年份筛选）
- 学术视野：本体设计、知识图谱可视化、SPARQL 查询演示、JSON-LD 语义标注
- 关于我们：项目介绍、技术架构、数据来源

## 技术栈
- HTML5/CSS3 (Flexbox/Grid 布局，响应式设计)
- JavaScript (ECharts 知识图谱，交互式查询)
- JSON-LD 结构化数据 (schema.org)
- 字体：Google Fonts (Inter, Noto Serif SC)

## 数据统计
- 9部电影作品
- 38项荣誉奖项
- 40+ 合作演员
